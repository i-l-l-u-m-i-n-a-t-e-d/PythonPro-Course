Polecenia zostały wykonane w PowerShellu:

python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install Django djangorestframework djangorestframework-simplejwt djoser
django-admin startproject config .
python manage.py startapp api
python -m pip freeze > requirements.txt

W pliku requirements.txt zostały zapisane wszystkie pakiety zainstalowane w środowisku. Najważniejsze z nich to:

Django==6.0.7
djangorestframework==3.17.1
djangorestframework_simplejwt==5.5.1
djoser==2.3.3
PyJWT==2.13.0
